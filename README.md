# Brew Haven ☕
An interactive, high-conversion landing page for a premium coffee subscription service. 

This project was developed as a part of a Front-End Development Internship at **Navodita Infotech**, focusing on mastering framework-free web technologies, mobile-first design, and web accessibility.

## 🛠️ Tech Stack
* **HTML5:** Semantic markup and ARIA accessibility standards.
* **CSS3:** Advanced Grid/Flexbox layouts, custom variables, and GPU-accelerated animations.
* **Vanilla JavaScript (ES6+):** Custom DOM manipulation, scroll-tracking interactions, and modal state management.
* **Backend:** Serverless form processing via Netlify Forms.

## ✨ Key Features
* **Mobile-First Responsive Design:** Fluid adaptability across all device breakpoints.
* **Scroll-Triggered Animations:** Smooth, 60fps progressive element reveals using `getBoundingClientRect()`.
* **Interactive UI/UX:** Unified hover states, custom modal overlays, and a dynamic glass-morphism navigation bar.
* **Serverless Backend:** Fully functional lead-capture contact form.
* **Performance Optimized:** Achieved a perfect 100/100 Best Practices Lighthouse score.
